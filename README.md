Address Book System (C)

Description

This is a simple Address Book application written in C.
It allows users to store, search, update, and delete contact details.

Features

* Add new contacts
* Search contacts
* Edit contact details
* Delete contacts
* File storage for data persistence

Technologies Used

* C Programming
* File Handling
* Basic Data Structures

How to Run

```bash
gcc src/*.c -o address_book
./address_book
```
